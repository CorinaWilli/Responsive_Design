# Responsive_Design

Bob the Square Swimmer

Dieses Projekt transformiert einen sachlichen Wikipedia-Eintrag über den Kofferfisch in einen modernen, spielerischen OnePager. Mittels Scrollytelling-Elemnten werden interessante Fakten interaktiv und spielerisch vermittelt, während ein durchdachtes und reduziertes Design-System für eine verständliche Navigation sorgt.

Zum OnePager: https://corinawilli.github.io/Responsive_Design/26-03-26-FINAL/index-V3.html
